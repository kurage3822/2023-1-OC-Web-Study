# 과제

## git의 명령어 branch, checkout, push

* branch

    ```
    git branch test
    ```

    test 브랜치를 추가

* checkout

    ```
    git checkout test
    ```

    test 브랜치로 전환

* push

    ```
    git push origin test
    ```

    리모트 리포지토리의 test 브랜치에 push (단, 리모트 리포지토리에 test 브랜치가 있어야 함)

## 느낀 점

브랜치 전환 후 작업과 리모트 리포지토리의 브랜치 유무 확인에 주의해야 함
